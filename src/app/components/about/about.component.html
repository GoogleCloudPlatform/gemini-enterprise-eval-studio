Gemini Enterprise Eval Studio

Overview

Gemini Enterprise Eval Studio is a specialized tool for cloud engineers and AI practitioners to rigorously test, evaluate, and compare responses from Google Cloud Discovery Engine (Vertex AI Search & Conversation). It leverages Gemini as an automated rater to provide objective semantic similarity scores and user defined rubrics along with the latency metrics.

Getting Started

1. Prerequisites

  • Google Cloud Access Token: You can get this by running gcloud auth print-access-token in your terminal. Note: tokens expire, so you may need to refresh them.
  • Gemini API Key: Obtain a key from the Google AI Studio.
  • Discovery Engine Setup: Ensure you have engines configured in your Vertex AI Search project.

2. CSV Formats

For Run Evaluation

Your CSV must contain at least these two columns:

  • query: The search or conversation prompt.
  • golden: The expected ideal response (baseline).

For Get Golden

Your CSV only needs:

  • query: The search or conversation prompt.

3. Metric Definitions

  • TTFT (Time to First Token): Latency until the first chunk of data (text or thought) is received from the model.
  • TFUFT (Time to First User Facing Token): Latency until the first actual text response (excluding internal chain-of-thought) is received.
  • Latency: Total time taken for the entire response to stream.
  • Score: A semantic similarity score (0.0 to 1.0) calculated by Gemini comparing the fetched response to your golden response.

4. Interpretation of Results

  • Delta (Δ): Calculated as Recent Score - Baseline Score.
  • Improvement (Green): Delta > +0.10.
  • Regression (Red): Delta < -0.10.
  • Stable (Gray): Changes within +/- 0.10 range.

Data Privacy & Governance

All sensitive tokens and keys are stored only in your browser's local storage for your convenience. They are never sent to any server other than directly to Google's official APIs during execution.

Gemini Enterprise Eval Studio is designed with a strict client-side, stateless architecture:

  • GCP Tenant Isolation: The tool operates entirely within your Google Cloud Platform (GCP) tenant. All computations, evaluation runs, and data storage occur within your controlled environment.
  • No Google Data Collection: Google does not collect, store, or have access to your customer data, queries, evaluation inputs, or evaluation results processed by this tool.
  • Governing Agreements: Any data handling and API calls made by the tool are governed solely by your existing agreements with Google Cloud for the specific APIs used (e.g., Vertex AI APIs).
